About Simpester

Elite Control is a single Flask + pywebview shell that hosts each of your tools as a module. Tools run as background jobs and stream their logs into the UI. Everything runs locally on your machine — nothing is sent anywhere except the sites each tool already talks to.

Adding more tools

Each module is two pieces: an adapter under control_center/modules/ exposing a run_x(job, params) function, and an entry in the MODULES list in app.py. The job object gives you add_log() and set_stats() so any tool gets live progress for free.
